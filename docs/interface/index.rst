pySTXM Main Screen
==================

.. figure:: /resources/images/interface_8_regions.png

The Main screen is divided up into 8 regions

	- 1 :doc:`/interface/window_system/counts`
		pySTXM's live counts and shutter control.
	- 2 :doc:`/interface/window_system/scan_plugin_prefs`
		This pane contains 3 sub panes, 
			- :doc:`/interface/window_system/scan_plugins/index`
			- :doc:`/interface/window_system/ioc_apps/index`
			- :doc:`/interface/window_system/prefs/index`
      
	- 3 :doc:`/interface/window_system/scan_ctrl`
		Take a tour and find out how to giver
	
	- 4 :doc:`/interface/window_system/data_visualization/index`
		The place where your data gets ...visualized
	
	- 5 :doc:`/data/index`
		nxSTXM definition of the Nexus file format
	
	- 6 :doc:`/devices/index`
	    Devices panel
	
	- 7 :doc:`/info/index`
	    Information panel
	
	- 8 :doc:`/sts_bar/index`
	    The status bar
